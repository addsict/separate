separate
===
Each command, each window

![gif](https://raw.github.com/addsict/separate/master/img/separate.gif)

Usage
---
```sh
separate [options] [FILE]
```

Options
---
```
-h       show help message
-v       separate vertically
```

Requirement
---
[tmux](http://tmux.sourceforge.net/) must be installed.
